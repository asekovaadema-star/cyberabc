# Cyber ABC

This is a mobile application developed for the Technovation Girls 2026 competition.
Our app is an educational game designed for children aged 5–10 to teach them the basics of cybersecurity in a fun and engaging way.

## Key Features

- **Postal Forest (Почтовый лес):** A level dedicated to email safety and recognizing phishing.
- **Social Jungle (Социальные джунгли):** A level about privacy and safe communication on social media.
- **Interactive Quizzes:** Gamified tests to reinforce learning.
- **Kid-Friendly UI:** Bright, intuitive design created specifically for younger users.

## Tech Stack

- **Platform:** Lovable, Kodular
- **Design:** Canva
- **Assets:** AI-generated images and custom graphics.
- **Logic:** Block-based coding / JavaScript / Python (json)

## Repository Structure

- `Public/` — Folder containing original icons and backgrounds.
- `Src/` — Main components and code.

## How to Run

To test the app on a phone, use the `.apk` file provided in the submission.
